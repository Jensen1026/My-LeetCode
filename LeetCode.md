   * 定义状态
   * 找到状态转移方程
2. **问题：**
   * Max/Min
   * Yes/No
   * Count(*)
   * Can't sort/swap
3. **四要素：**
   * 状态 state
   * 初始化 init
   * 方程 function
   * 结果 result
4. 常见类型
   * 矩阵DP
   * 序列DP
   * 双序列 DP
   * 背包问题
